# Getting Started

# Get It and Build It
```bash
# Get the repo
$ git clone https://github.com/emilitronic/Sequelizer.git
# Go in the repo
$ cd Sequelizer
# Make a place for the executables
Sequelizer $ mkdir build && cd build
# Build it
Sequelizer/build $ cmake .. && cmake --build .
# See it
Sequelizer/build $ ls
... sequelizer* ...
```
